# QA Projects

This repository contains sample QA projects for learning and demonstration purposes.

## 📋 Projects Included
1. **Manual Testing** – OpenCart Web Application  
   - Test cases written in Excel  
   - Covers Login, Search, Add to Cart, and Checkout modules  

2. **API Testing** – Using Postman  
   - Tested endpoints like /login, /products, and /cart/add  
   - Methods used: GET and POST  

3. **Performance Testing** – Using JMeter  
   - Load tested the product listing API with 5 virtual users  
   - Measured response time and success rate  

## 🧰 Tools Used
- Selenium WebDriver with Java  
- Postman  
- JMeter  
- Excel  
- Chrome DevTools  

## 📄 Notes
These projects are created for practice and demonstration of QA skills.
